prompt 和 instruction 有什么区别？
简单说，prompt 是你给模型的完整输入，而 instruction 是其中用来告诉模型“要做什么”的指令。

例如：
“请用一句话解释 token” 是 instruction；连同背景、示例一起提交给模型的整段内容，才是 prompt。
