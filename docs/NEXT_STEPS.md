# 下一步执行清单

更新日期：2026-06-25

## 本周优先级

1. 接入 Azure Speech
   - 用户创建 Azure Speech Free F0 资源；
   - 在后台填写 Key、Region、Voice ID；
   - 服务端实现真实 TTS；
   - 生成音频并转换为 SILK；
   - 发送给企微用户测试。

2. 固定第一版知识点格式
   - 主题：AI 行业常用英语；
   - 每条内容包含：单词、中文解释、英文例句、使用场景、一个互动问题；
   - 先做 20 条种子内容。

3. 完善推送策略
   - 默认每日一条；
   - 支持后台配置推送时间；
   - 支持文本-only / 文本+语音；
   - 支持手动测试推送。

4. 验证 10 个真实用户
   - 添加员工号；
   - 自动订阅；
   - 收到知识点；
   - 完成一次互动；
   - 收到语音。

## 交付验收标准

- 后台可以配置语音；
- 用户可以收到真实 TTS 生成的企微语音；
- 每条推送有日志；
- 失败时不重复骚扰用户；
- 不泄露任何密钥到前端或 GitHub；
- 可以稳定连续推送 3 天。
